#Нейронные_сети_для_МЭ_эффекта

Эта директория содержит проект по использованию нейронных сетей (НН) для исследования магнитоэлектрического (МЭ) эффекта. Здесь предложены три способа применения НН. Для каждого из них своя директория.
Первая (директория "DW bend position") относится к применению НН для определения позиции и кривизны изгиба доменной границы (ДГ).
Вторая (директория "Type of ME mechanism") относится к применению НН для классификации механизмов МЭ эффекта, запечатлённого на парах магнитооптических фотографий.
Третья (директория "Mythic ME places") относится к поиску аномальных мест на магнитооптических фотографиях, где МЭ эффект отличается от обычного МЭ эффекта для данного образца.
Подробная информация находится в каждой директории
